# Dao Learning Lab

把具体知识的“术”沉淀为可迁移的“道”，并用小项目验证模型。

## 目录

- `knowledge-trees/`：问题、关键追问、因果链与边界的 Mermaid 知识树。
- `dao/`：跨领域可迁移的抽象模型。
- `projects/`：从知识迁移得到的最小可验证项目。
- `templates/`：后续学习记录与项目的模板。
